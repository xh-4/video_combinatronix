    • PROJECT: Resonates with feature richness

┌─────────────────────────────────────────────────────────────────────────────┐
│                            DATA FLOW PATTERN                               │
└─────────────────────────────────────────────────────────────────────────────┘

    Input Data → FieldIQ Conversion → Stage Processing → Output → Reset → Repeat

    Data Sources:
    • Video frames (VideoFrame)
    • Video chunks (VideoChunk) 
    • Numpy arrays
    • Synthetic data (for demo)

    Processing:
    • Each stage uses composed activation realms
    • Resonance determines stage transitions
    • Carriers run stages in parallel
    • Continuous cycle with state management

    • Resonance scheduling through VM combinators

This architecture enables:
✓ Continuous AI processing pipelines
✓ Resonance-based stage selection  
✓ HPU carrier load balancing
✓ Real-time data ingestion
✓ Scalable activation realm composition
✓ VM integration for cross-language deployment
